Mac下Hugo的安装和使用

安装Hugo

  • 不要使用brew来安装,得到的不是最新版本
  • 推荐使用tar方式安装,去releases page 下载mac os对应的版本
  • 解压缩放到/usr/local/bin下.

生成静态html

  • git page 目前都是使用https所以在config.toml里面设置baseurl = “https://atreeyang.github.io/"
  • 使用命令 hugo –buildDrafts
  • 去public目录 提交到你自己的git仓库,访问即可
